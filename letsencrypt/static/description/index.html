Let’s Encrypt Modul

Denna modul hanterar automatiserad SSL-certifikatutfärdande från Let's Encrypt för din Odoo-installation.

Efter installation genereras en privat nyckel automatiskt och certifikat uppdateras med en cronjob.

Modulen kräver att webbplatsen är tillgänglig via HTTP (port 80) för att validera domänen.

För konfiguration och användning, se modulens dokumentation i Odoo.
